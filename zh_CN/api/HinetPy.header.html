HinetPy.header模块

class HinetPy.header.Network(name, channels, starttime, url)

包含了台网信息的类。

name

台网名。

channels

此台网的通道数目。

starttime

可申请的波形数据的开始时间（日本标准时间）。

url

台网主页。

NETWORK 是一个 Network 词典，包含了Hi-net网站上所有可用的台网的信息。

>>> from HinetPy import NETWORK
>>> for code in NETWORK.keys():
...     print(code, NETWORK[code].name)
