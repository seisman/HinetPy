HinetPy.header module¶

class HinetPy.header.Network(name, channels, starttime, url)¶

An object containing information of a network.

name¶

Network name.

channels¶

Number of channels the network has.

starttime¶

Start time (JST: UTC+0900) when waveform data is avaiable.

url¶

Homepage of the network.

NETWORK is a dict of Network, containing information of all networks available from Hi-net website.

>>> from HinetPy.header import NETWORK
>>> for code in NETWORK.keys():
...     print(code, NETWORK[code].name)
0101 NIED Hi-net
0103 NIED F-net (broadband)
0103A NIED F-net (strong motion)
0120 NIED S-net (speed motion)
0120A NIED S-net (acceleration)
0131 NIED MeSO-net
0201 Hokkaido University
...
